# Automated-Paper-Evaluation-System

## 📌 Overview

The **Automated Paper Evaluation System** is an AI-powered web application that evaluates handwritten exam papers in PDF format. It uses the **Google Gemini API** to analyze answers and provide automated grading, reducing manual effort and improving evaluation speed.

## 🚀 Features

* 📄 Upload handwritten exam papers in PDF format
* 🤖 AI-based answer evaluation using Google Gemini API
* ✅ Automatic Pass/Fail result generation
* 📊 Section-wise performance analysis
* 📝 User-friendly web interface

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **AI Integration:** Google Gemini API
* **PDF Processing:** PDF.js

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/sandhyaR7/Automated-Paper-Evaluation-System.git
   ```
2. Open the project folder.
3. Configure your Gemini API key if required.
4. Open `index.html` in your browser.
